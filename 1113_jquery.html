제이쿼리란?

Javascript가 하는 일은 HTML과 CSS요소를 선택한 뒤 동적으로 수정하여 웹화면에 인터렉션을 주는게 주목적이다.
jQuery 이용하면 로딩이 완료된 웹페이지에 사용자가 원하는 행동에 따라 쉽게 인터렉션을 나타내게 할 수 있는 라이브러리 형태이다.

작성법

  1. 다운로드 받지 않고 링크연결을 통해 최신버전에 해당하는 코드를 연결한다. 이를 CDN이라고 부른다.
    http://code.jquery.com/jquery-latest.js를 script src에 연결한다.
  2. http://jquery.com에 접속하여 파일을 다운로드하여 폴더에 저장 후 연결하여 사용한다.
  3. 링크연결을 완료한 후
    $(document).ready(function(){
    실행문작성 기재
    })
  4. $(function(){
    실행문작성 기재
    })

제이쿼리 선택자

$('선택할 요소') : 클래스 - .선택자, id - #선택자, 태그 - 태그명
변수는 ''(따옴표)안에 넣지 않음

제이쿼리 상세선택자

  • 자식요소 선택 : .children('선택자이름')
  • 부모요소 선택 : .parent('선택자이름')
  • 순서값 설정 : .eq(순번 0부터) 요소의 순서
  • 다음요소 .next()
  • 이전요소 .prev()
  • 형제요소 선택 : .siblings() 나를 제외한
  • 자손요소 선택 : .find('선택자이름')
  • 특정요소를 제외하기 : .not('선택자이름')

이벤트

실제 사용자가 웹상에서 특정한 행동을 취했을 때 그에 대한 상호작용이 나타나는 형태

  • 클릭 - .click(function(){실행문})
  • 마우스를 올렸을때 - .mouseenter(function(){실행문})
  • 마우스를 내렸을때 - .mouseleave(function(){실행문})
  • 마우스가 움직였을때 - .mousemove(function(){실행문})
  • 브라우저를 스크롤했을때 - $(window).scroll(function(){실행문})
  • 브라우저를 리사이즈했을때 - $(window).resize(fucntion(){실행문})
